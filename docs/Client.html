Class: Client

Client

new Client(serverKey, options)

The constructor for the Client object. This is the main entry point to the standard Postmark API.

var client = new Client('<server key>');

// You can now use any of the methods associated with the client.
Parameters:
Name Type Argument Description
serverKey string

The server key is required, and specifies which Server you wish to use when making API calls from this Client.
options ClientOptions <optional>

The configuration options for this client. If not specified, the global defaults will be used.
Returns:
